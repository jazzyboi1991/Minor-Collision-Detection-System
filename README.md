# 🚗 물피도주 감지 시스템 (Minor Collision Detection System) - Frontend

본 리포지토리는 물피도주(주차 중 사고 후 도주) 사례를 지능적으로 감지하고 분석하는 시스템의 웹 서비스 프론트엔드 결과물을 담고 있습니다.

## 📋 프로젝트 개요

이 시스템은 CCTV 및 차량 블랙박스 영상을 분석하여 차량 간의 충돌 발생 여부를 감지하고, 사용자에게 시각화된 데이터와 분석 결과를 제공하는 웹 인터페이스를 포함합니다.

## 📂 폴더 구조 및 파일 설명

- **`src/`**: React 소스 코드가 포함된 핵심 디렉토리입니다.
- **`public/`**: 이미지, 아이콘 등 정적 자산이 포함되어 있습니다.
- **`AI-algorithm/`**: 영상 분석 및 데이터 처리를 위한 알고리즘 관련 스크립트가 포함되어 있습니다.
- **`index.html`**: 서비스의 메인 진입점인 HTML 파일입니다.
- **`vite.config.js`**: Vite 빌드 및 개발 서버 설정 파일입니다.
- **`package.json`**: 프로젝트 의존성 및 스크립트 정의 파일입니다.

> [!NOTE]
> `Data/` 폴더는 대용량 영상 및 학습 데이터를 포함하며, `.venv/`는 파이썬 가상 환경으로 Git 추적 및 프론트엔드 빌드 대상에서 제외됩니다.

## 🚀 실행 방법 (Vite)

본 프로젝트는 최신 프론트엔드 도구인 **Vite**를 사용하여 개발되었습니다. 특히 외장 하드(ExFAT) 환경에서의 원활한 실행을 위해 아래의 특수한 실행 방법을 권장합니다.

### 1. 의존성 설치

심볼릭 링크를 지원하지 않는 외장 하드 환경을 고려하여 아래 명령어로 패키지를 설치합니다.

```bash
npm install --no-bin-links
```

### 2. 개발 서버 실행

일반적인 `npm run dev` 명령어가 작동하지 않을 경우, 아래와 같이 Vite 실행 파일을 직접 호출하여 서버를 실행할 수 있습니다.

```bash
node node_modules/vite/bin/vite.js
```

실행 후 터미널에 표시되는 로컬 주소(예: `http://localhost:5173`)를 브라우저에서 열어 서비스를 확인할 수 있습니다.

## 🛠 기술 스택

- **Framework**: React 19
- **Build Tool**: Vite 8
- **Styling**: CSS (Vanilla)
- **Language**: JavaScript (ESM)
